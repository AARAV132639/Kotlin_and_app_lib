# Combined Basic Insights of Phase 1 implementations: Kotlin Fundamentals

## Introduction

1. *Kotlin* is a modern, statically typed programming language developed by JetBrains

2. Runs on JVM and is fully interoperable with Java and is preferred language for Android App developement

## Basic Points

- *Null Safety* : Built-in protection against **NullPointerException** (one of Java's biggest pain points)

- Can use all Java libraries and could be called from Java seamlessly

- Google recommends Kotlin for new Android projects

- *Kotlin Multiplatform* (KMP): allows sharing business logic across Android, iOS, Web, Desktop and even servers- while keeping native performance

## Fun facts

- Semi colons are optional and type declarations come after the name (like in typescript/rust) for better readability in complex cases

- Blends object-oriented and functional programming elegantly while staying pragmatic [Refer Principles of Programming Language]
